## Summary

Get the absolute position of an element.

## Syntax

```c
keyman.util.getAbsolute(Pelem);
```

### Parameters

`Pelem`
:   Type: `Element`
:   An HTML element.

### Return Value

`Object`
:   An object with two `number` properties (`x` and `y`) corresponding to the specified element's absolute position on the page.
