title: linkStyleSheet
---

## Summary

Add a reference to an external stylesheet file.

## Syntax

```c
keyman.util.linkStyleSheet(s);
```

### Parameters

`s`
:   Type: `string`
:   Path to a pre-existing stylesheet file.

### Return Value

`undefined`
