## Summary

Get the absolute x-coordinate of an element.

## Syntax

```c
keyman.util.getAbsoluteX(Pelem);
```

### Parameters

`Pelem`
:   Type: `Element`
:   An HTML element.

### Return Value

`number`
:   The specified element's absolute x-coordinate on the page.
