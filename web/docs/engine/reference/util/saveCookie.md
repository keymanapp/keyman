## Summary

Save document variables as a cookie.

## Syntax

```c
keyman.util.saveCookie(cn, cv);
```

### Parameters

`cn`
:   Type: `string`
:   Name of the cookie to save.

`cv`
:   Type: `Object`
:   An array of named arguments and values.

### Return Value

`undefined`
