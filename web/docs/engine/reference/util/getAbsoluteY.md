## Summary

Get the absolute y-coordinate of an element.

## Syntax

```c
keyman.util.getAbsoluteY(Pelem);
```

### Parameters

`Pelem`
:   Type: `Element`
:   An HTML element.

### Return Value

`number`
:   The specified element's absolute y-coordinate on the page.
