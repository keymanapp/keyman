---
title: show
---

## Summary

Display, hide or toggle OSK visibility.

## Syntax

```c
keyman.osk.show(mode);
```

### Parameters

`mode`
:   Type: `boolean` *optional*
:   `true` to display, `false` to hide, omitted to toggle.

### Return Value

`undefined`

## Description

Visibility will be toggled on or off if the argument is omitted.
