---
title: ShowPinnedHelp
---
  
## Summary

Shows the on-screen keyboard and locks it at the resulting location.

## Syntax

```
keyman.ShowPinnedHelp(x, y);
```

### Parameters

None.

### Return Value

`undefined`

### Replaced by

[`keyman.osk.setRect()`](../osk/setRect),
[`keyman.osk.show()`](../osk/show)
