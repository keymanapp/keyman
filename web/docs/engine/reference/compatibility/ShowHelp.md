title: ShowHelp
---
  
## Summary

Shows the on-screen keyboard at the requested (x, y) coordinates.

## Syntax

```
keyman.ShowHelp(x, y);
```

### Parameters

`x`
:   Type: `number`
:   x-coordinate for display of the OSK.

`y`
:   Type: `number`
:   y-coordinate for display of the OSK.

### Return Value

`undefined`

### Replaced by

[`keyman.osk.setPos()`](../osk/setPos),
[`keyman.osk.show()`](../osk/show)
