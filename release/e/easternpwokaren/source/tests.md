# User Test Results

## Test 1: Desktop / Web Layout
1. Install the keyboard layout or open the Keyman Web testing page.
2. Switch your active keyboard to **Eastern Pwo Karen**.
3. Type the following keys on your hardware keyboard: `abc`
4. Verify that the characters displayed on the screen look correct for Eastern Pwo Karen: `ပ်ုဖၠုံဖေါဟ် အှ်ၰူ့အှ်ၯင်`

## Test 2: Touch / Mobile Layout (If applicable)
1. Open the keyboard layout on a mobile simulator or the Keyman app.
2. Select the **Eastern Pwo Karen** layout.
3. Tap the keys to verify the layout matches your design.
4. Long-press keys to verify that any pop-up/hidden alternate characters appear correctly.
