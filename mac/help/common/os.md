## Supported Mac Operating Systems

Keyman supports the following Mac operating systems:

* macOS High Sierra (10.13)
* macOS Mojave (10.14)
* macOS Catalina (10.15)
* macOS Big Sur (11.0)
* macOS Monterey (12.0)
* macOS Ventura (13.0)
* macOS Sonoma (14.0)
