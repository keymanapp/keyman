Help files for JVCL (the "H-Files")

Coordinator: Marcel Bestebroer

As the tool for creation of JVCL Help files we use Doc-O-Matic, , graciously donated by ToolsFactory. 

But you don't need to use Doc-O-Matic to help us with creating Help files for JVCL. You can submit simple text files following the guidelines for JVCL Help writers.
