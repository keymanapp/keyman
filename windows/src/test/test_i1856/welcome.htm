Issue Details

The welcome.htm page often needs to link to other pages (html or pdf) that are included in a package. If the link uses the same window, there is no way to return to the original welcome page window (unless a specific link is included in each document). But if target='_blank' is used to open a new window, IE crashes if the window is new window is closed before the original window. Also, the new window is not created as a topmost window, so is hard to read anyway without.

The best solution (IMHO) would be to add a Home (or Back) link on the bottom of the IE window that always either goes back to the previous page or back to the main welcome.htm page.
