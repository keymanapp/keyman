Contact and Support

The JVCL is built by the community, but more importantly, for the community. Therefore we would like to hear from you. If you have any problems, comments or suggestions we would appreciate it if you would drop us a note. There are several ways to get in contact with us:

  • You can submit a bug or feature request via our Bug/Issue Tracker 

  • If you want to keep up to date about JVCL then you can join the JVCL mailing list by entering your email address below, and pressing the JVCL button. You can also use this list to voice your opinion, comments or suggestions.

To access SourceForge Forums via Newsgroups, please configure your newsgroup reader as follow:

  • - The hostname of the NNTP server is 'nntp.sourceforge.net'
  • - the port NNTP is listening on is 563.
  • - nntps, SSL-protected NNTP (SourceForge allows only SSL connections)

The names of the JVCL newsgroups are:

  • sf.j.jv.jvcl.help 
  • sf.j.jv.jvcl.open_discussion 

You will need to use your SourceForge user name and password to login. If you don't have one, please go to:

http://sourceforge.net/account/register.php 

and register.

The currently supported clients are Mozilla and Outlook Express, however SourceForge has followed the NNTP RFC; so any client that does NNTP over SSL "should" work.
