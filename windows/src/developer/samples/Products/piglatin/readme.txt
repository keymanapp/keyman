This sample demonstrates a customised user interface for Keyman Desktop
with a simple Pig Latin keyboard (pig latinisation limited to 6 characters).

* piglatin.kmn: a sample keyboard for Pig Latin - have fun extending it!

* piglatin.kps: contains piglatin.kmx

* piglatin.kct: a branding file that customises the Keyman Desktop user interface

* piglatin.kpp: sample product source file
