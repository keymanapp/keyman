The samples in this folder demonstrate some of the features of the Keyman Keyboard
Language:

* enhlao.kmn: A complex example using a legacy code page, that includes
  syllable analysis for inserting syllable breaks, for Lao.

* ipamenu.kmn: A simple example of a menu-based keyboard.  May be useful for
  extended characters in some keyboards

* nepali.kmn: A basic keyboard for Nepali

* qfrench.kmn: The complete Quick French keyboard from the Keyboard Tutorial

* European\european.kmn: A complete European Latin keyboard layout

* European\european.kps: A comprehensive package with Welcome, Readme,
  a graphic, and other features
